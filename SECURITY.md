# Security Policy

RR-framework is an Excel-based methodology and diagnostic workbook. It may be used to collect sensitive organizational information, including risk, technology, operational, workforce, supply chain, and crisis-readiness data.

## Supported versions

| Version | Supported |
|---|---|
| 2.0 | Yes |
| < 2.0 | No |

## Reporting security issues

If you identify a security issue in the workbook, documentation, repository, or future integration logic, please report it privately to the repository maintainer rather than opening a public issue.

Recommended report content:

- Affected version.
- Affected sheet or file.
- Description of the issue.
- Steps to reproduce.
- Potential impact.
- Suggested fix, if available.

## Data handling warning

Do not upload completed client assessments, confidential questionnaire responses, personal data, or sensitive risk information to public GitHub issues, pull requests, or discussions.

Use anonymized examples only.

## Macro policy

The workbook is intended to work without macros and ships as a standard `.xlsx`. Users should be cautious with any modified version that introduces macros, external connections, or scripts.
