# Release Notes: RR-framework v1.0

Release date: 2026-06-07

Build ID: RR-FRAMEWORK-v1.0-20260607

## Summary

This is the initial public release of RR-framework, an Excel-based business resilience and digital maturity assessment workbook.

## Highlights

- Company Profile with closed-answer inputs.
- Domain questionnaires across organizational functions.
- Process maturity, digitalization, and resilience scoring.
- Industry template logic.
- Crisis scenario simulation.
- Risk summary and risk library.
- Executive recommendations.
- Executive strategy packages.
- Initiative roadmap and Gantt roadmap.
- Heatmaps and spider dashboards.
- Standards mapping per question.
- Quality control layer.
- Copyright, license, footer, custom document properties, and version history.
