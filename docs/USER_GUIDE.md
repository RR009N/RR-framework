# User Guide

## Purpose

RR-framework v3.0 guides a user from company context to assessment, scenario analysis, risks, strategies, initiatives, and roadmap.

## Recommended workflow

1. Open `workbook/RR-framework_v3.0.xlsx`.
2. Start on **Guidelines**.
3. Fill **Company Profile**.
4. Complete **Assessment Questions**.
5. Select crisis scenario and severity in **Resilience & Risk Scenario**.
6. Review **AS-IS Assessment**.
7. Review **Action Plan & Roadmap**.
8. Recalculate in Microsoft Excel before exporting formal results.

## Editable areas

Edit visible input fields only. Hidden `_DATA_*`, `_ENGINE_*`, `_CONFIG_*`, `_REF_*`, `_API_*`, `_README`, `_LINT`, and `_DOC` sheets are model/backend/documentation layers.

## Recalculation

Before saving a final report:

```text
Excel → Formulas → Calculation Options → Automatic
Then press Ctrl+Alt+F9
Save the workbook
```
