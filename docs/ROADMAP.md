# Roadmap

This roadmap describes possible future development of RR-framework.

## Version 1.x

Focus: Excel workbook stabilization and documentation.

Planned improvements:

- Improve client-facing Executive Output page.
- Add scenario comparison view.
- Add triggered-risk view for selected crisis scenarios.
- Improve recommendation clustering into strategy packages.
- Improve workbook usability and navigation.
- Add sample anonymized assessment data.
- Add more industry templates.

## Version 2.x

Focus: Forms-based data collection and controlled assessment workflow.

Planned improvements:

- Google Forms or Microsoft Forms integration guide.
- Respondent assignment model.
- Raw response and normalized response data model.
- Multi-respondent aggregation logic.
- Disagreement flags.
- Evidence gap flags.
- Client Edition and Consultant Edition separation.

## Version 3.x

Focus: Application architecture.

Possible future direction:

- Web-based assessment interface.
- Database-backed response storage.
- Role-based access control.
- Automated dashboards.
- Excel export and PDF export.
- API for calculations and integration.

## Not planned for the core public workbook

The following may remain optional or separate:

- Commercial consulting templates.
- Client-specific benchmarks.
- Confidential assessment datasets.
- Formal certification workflows.
- Financial forecasting or investment-grade valuation logic.
