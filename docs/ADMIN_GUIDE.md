## Administration principles

- Do not edit engine formulas directly unless doing controlled model development.
- Keep version metadata aligned across Guidelines, `_CONFIG_Model_Settings`, `_README`, `_DOC`, `_LINT`, core properties and custom properties.
- Preserve attribution, DOI, ORCID and license metadata.
- Use `_LINT` and static XML scans after structural workbook changes.
- Test workbook opening/recalculation in Microsoft Excel desktop before publication.
